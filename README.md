# AuthX

A secure-authentication project developed for the course **Dezvoltarea Aplicațiilor Software Securizate**.

## Overview
AuthX is a Java Spring Boot application focused on:
- authentication
- session management
- password reset
- ticket CRUD
- audit logging

The project includes:
- a **vulnerable version (v1)**
- a **secured version (v2)**

## Main topics covered
- weak password policy
- insecure password storage
- brute force / lack of rate limiting
- user enumeration
- weak session handling
- weak password reset tokens

## Tech stack
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Postman
- IntelliJ IDEA

## Purpose
The goal of the project is to demonstrate common authentication vulnerabilities and then remediate them in a secured version of the application.
